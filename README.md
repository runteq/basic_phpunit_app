## 環境構築
1. リポジトリをfork
2. `git clone`して`cd basic_phpunit_app`で移動
3. ブランチを各課題に応じて作成＆移動（参考：　git checkout -b ブランチ名）
4. docker compose build, docker compose upコマンドを実行
5. phpunitファイルを修正して、テスト実行
